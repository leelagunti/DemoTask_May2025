# Shielded SDET Automation Demo

This project demonstrates automated UI testing using **Playwright** and **TypeScript** for the [Shielded.io](https://shielded.io) website.

It includes:
- Homepage validation
- Careers page
- Page Object Model design


---

## Test Scenarios

1. **Homepage Test**  
   - Validates presence of main headlinesection.

2. **Career Page Test**  
   - Searches for “Software Development Engineer in Test”
   - Check Role is Remote
   - Check Role is Full time
